# Roadmap

## Fase 1: Base del portfolio

- Sitio Astro estático.
- Contenido en español.
- Páginas principales.
- Portafolio con colección tipada.
- Galerías por proyecto con carrusel y lightbox.
- Filtros de portafolio por categoría.
- Paquetes comerciales en servicios.
- README y documentación inicial.
- Suite end-to-end con Playwright para rutas e interacciones principales.

## Fase 2: Producción visual

- Reemplazar placeholders por material propio.
- Crear versiones optimizadas de imágenes.
- Añadir video hero real o showreel hospedado si el hosting lo permite.
- Refinar dirección de arte con screenshots reales.

## Fase 3: Conversión y contacto

- Conectar formulario a un servicio real.
- Añadir confirmación por correo.
- Registrar eventos de conversión.
- Mejorar preguntas condicionales según tipo de servicio.

## Fase 4: Confianza y crecimiento

- Añadir más casos de estudio.
- Publicar testimonios verificables.
- Agregar datos estructurados.
- Añadir blog o notas de proceso si aporta a SEO.

## Fase 5: Calidad continua

- Pruebas visuales.
- Auditoría de accesibilidad.
- Presupuesto de performance.
- Revisión periódica de dependencias.
