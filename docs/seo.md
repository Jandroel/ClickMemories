# SEO

ClickMemories usa una base SEO estática orientada a búsquedas locales y servicios audiovisuales.

## Implementación actual

`BaseLayout.astro` define:

- Título por página.
- Descripción.
- Canonical.
- Open Graph.
- `twitter:card`.
- Idioma `es`.
- Theme color.

También existen:

- `public/robots.txt`.
- `src/pages/sitemap.xml.ts`.
- URLs limpias para páginas principales y proyectos.

## Recomendaciones de contenido

Cada proyecto debe tener un título claro, categoría, ubicación, resumen y contenido que describa intención, proceso y entregables. Esto ayuda a que el portfolio no sea solo visual, sino también entendible por buscadores y clientes.

## Antes de publicar

- Reemplazar `https://clickmemories.example.com` por el dominio real.
- Revisar títulos y descripciones finales.
- Añadir datos estructurados si se definen servicios y ubicación comercial definitiva.
- Conectar Search Console cuando el dominio esté activo.
