# Sistema de diseño

El sistema visual de ClickMemories busca una sensación editorial: elegante, cálida, cinematográfica y humana. La interfaz no compite con las fotografías; las acompaña con jerarquía clara y composición contenida.

## Voz de marca

La voz debe sonar refinada y emocional, sin exceso de solemnidad. Se evitan frases genéricas como “capturamos tus mejores momentos” cuando no aportan intención. La escritura prioriza memoria, luz, detalle, presencia y narrativa visual.

## Paleta

- `#161513`: tinta principal, fondos oscuros y contraste.
- `#fbfaf7`: papel claro de base.
- `#f2eadc`: crema para bandas secundarias.
- `#a5573a`: arcilla para énfasis y mensajes.
- `#1f5146`: verde profundo para datos destacados.
- `#3d5a80`: azul sobrio para metadatos.
- `#d9a441`: dorado para acentos pequeños.

La paleta evita depender de un solo color. Los acentos aparecen con moderación para que la fotografía conserve protagonismo.

## Tipografía

Se usa una combinación de serif del sistema para titulares y sans del sistema para interfaz. Esta decisión reduce dependencias y mantiene una apariencia editorial.

Los tamaños se ajustan por puntos de quiebre, no con escalado continuo por viewport. La legibilidad móvil tiene prioridad sobre titulares sobredimensionados.

## Componentes

- Header con navegación principal y menú móvil nativo.
- Hero con imagen de fondo, marca visible y CTAs.
- Tarjetas de proyecto con imagen, categoría, descripción, contador de piezas y mini tira visual.
- Portafolio con portada inmersiva, filtros agrupados y grilla editorial asimétrica.
- Galería de proyecto con carrusel, miniaturas, swipe y lightbox.
- Tarjetas de servicio con precio inicial, descripción y entregables.
- Paquetes de servicio para orientar decisiones comerciales.
- Formulario con labels visibles y mensajes de validación en español.
- Bandas visuales full-width para sostener ritmo editorial.

## Responsive

El layout usa grids que pasan de 3 o 4 columnas a 2 y luego a 1 columna. Los elementos fijos como navegación, cards y formulario tienen dimensiones estables para evitar saltos visuales.
