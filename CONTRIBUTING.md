# Contribuir a ClickMemories

Gracias por ayudar a mejorar ClickMemories. Este repositorio prioriza claridad, contenido en español y una experiencia visual cuidada.

## Antes de empezar

- Revisa `README.md`.
- Revisa la documentación en `docs/`.
- Crea una rama en inglés y kebab-case.
- Mantén el contenido visible del sitio en español.

## Convenciones

Los commits deben usar `Conventional Commits` en inglés:

```text
feat: add services page
fix: improve mobile contact form
docs: update SEO notes
```

Los identificadores de código, nombres de archivo y scripts pueden mantenerse en inglés por convención técnica.

## Calidad

Antes de abrir un PR, ejecuta:

```bash
npm run build
npm run test:e2e
```

Si el cambio modifica UI, incluye capturas de escritorio y móvil.

## Contenido

El tono debe ser elegante, emocional, moderno y premium. Evita frases genéricas o demasiado comerciales. Prioriza detalles concretos de luz, memoria, intención y narrativa visual.
